> 작성일: 2026-02-14
> 스타일: Professional / 녹색 계열 강조 / 흰색 배경

---

## WHY: 프리젠테이션의 목적과 핵심 메시지

### 프리젠테이션 목적
**"개발자가 생성형 AI 어플리케이션을 실전에 투입할 수 있도록 체계적인 학습 경로를 제시한다"**

### 핵심 메시지 (3가지)

1. **실습 중심의 실전 교육**
   - 이론 40% : 실습 60%의 황금 비율
   - 매 모듈마다 hands-on 랩 제공
   - 즉시 업무에 적용 가능한 실무 패턴 학습

2. **기초부터 고급까지 완벽한 로드맵**
   - 기본과정(3일): API 호출부터 멀티모달 AI까지
   - 심화과정(4일): RAG, Agent, MCP까지 최신 기술 완전 정복
   - 단계별 난이도 설계로 학습 부담 최소화

3. **2025년 최신 트렌드 완벽 반영**
   - GraphRAG, Multi-Agent System
   - Model Context Protocol (MCP)
   - Dify 노코드 플랫폼
   - 비즈니스 즉시 적용 가능

---

## HOW: 전달 방식과 구성 전략

### 전달 전략

1. **문제-해결 구조**
   - 각 섹션: "왜 필요한가?" → "무엇을 배우나?" → "어떻게 적용하나?"
   - 구체적 비즈니스 시나리오 예시 활용

2. **시각적 계층 구조**
   - 기본/심화 과정을 색상으로 구분 (기본: 연녹색 / 심화: 진녹색)
   - 아이콘과 도표로 복잡한 개념 단순화
   - 3개 이하 핵심 포인트로 집중력 유지

3. **신뢰성 확보**
   - 구체적 수치 제시 (일정, 시간, 비중)
   - 실습 예시 스크린샷 포함
   - 평가 기준 투명 공개

### 슬라이드 구성 전략

- **총 15장 내외** (인사 제외)
- **3-Act 구조**: 도입(3장) → 본론(10장) → 마무리(2장)
- **한 슬라이드 = 한 메시지** 원칙 준수
