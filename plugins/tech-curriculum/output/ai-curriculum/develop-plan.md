## 2. 교육 목표

### 기본과정 목표

- LLM API의 핵심 개념(프롬프트, 토큰, 모델 파라미터)을 정확히 이해한다
- 멀티턴 대화, 문서요약, PDF 처리, 음성(STT/TTS), 비전(VLM) 등 주요 모달리티별 AI 어플리케이션을 직접 구현한다
- Cloud LLM과 Local LLM의 트레이드오프를 이해하고 비즈니스 상황에 맞는 선택 기준을 수립한다

### 심화과정 목표

- Function Call과 LangChain을 활용한 도구 연동 및 체인 구성 능력을 확보한다
- RAG 파이프라인을 설계하고 품질 튜닝(청킹, 임베딩, 리랭킹)을 수행한다
- Multi-Agent System, GraphRAG, MCP, Dify를 활용하여 실무 수준의 AI 에이전트를 구축한다

## 4. 전체 구성 개요

### 기본과정 (3일, 24시간)

| 일차 | 주제 | 핵심 내용 |
|------|------|-----------|
| Day 1 | AI 어플리케이션 기초 | AI 개요, 개발환경 구축, LLM API 기초, 멀티턴 대화 구현 |
| Day 2 | 문서 처리와 음성 AI | 문서요약, PDF 처리, STT, TTS, 파이프라인 연결 |
| Day 3 | 비전 AI와 Local LLM | VLM(이미지 이해), Local LLM 구축, Cloud vs Local 비교, 기본과정 종합 실습 |

### 심화과정 (4일, 32시간)

| 일차 | 주제 | 핵심 내용 |
|------|------|-----------|
| Day 1 | 도구 연동과 프레임워크 | Function Call, LangChain 기초와 체인 구성 |
| Day 2 | RAG 구축과 튜닝 | RAG 파이프라인, 품질 튜닝, 웹검색 통합 |
| Day 3 | 고급 RAG와 에이전트 | GraphRAG, Multi-Agent System 설계 |
| Day 4 | 통합과 실무 적용 | MCP 도구통합, Dify 노코드 Agent, 종합 프로젝트 |
