## 목표

tech-curriculum 플러그인의 사용 가능한 명령, 자동 라우팅 규칙, 주요 기능을 안내함.
런타임 상주 파일(CLAUDE.md)에 라우팅 테이블을 등록하는 대신,
이 스킬이 호출 시에만 토큰을 사용하여 사용자 발견성을 제공함.

[Top](#help)

---

## 활성화 조건

- `/tech-curriculum:help` 슬래시 명령 호출 시
- "도움말", "뭘 할 수 있어", "help" 키워드 감지 시

[Top](#help)

---

## 스킬 부스팅

이 스킬의 워크플로우는 `ulw` 매직 키워드를 활용하여 병렬 실행 + 완료 보장 방법론을 적용함.

[Top](#help)

---

## 명령어

| 명령 | 설명 |
|------|------|
| `/tech-curriculum:setup` | 플러그인 초기 설정 (도구 설치) |
| `/tech-curriculum:add-ext-skill` | 외부호출 스킬(ext-{대상플러그인}) 추가 |
| `/tech-curriculum:remove-ext-skill` | 외부호출 스킬(ext-{대상플러그인}) 제거 |
| `/tech-curriculum:help` | 사용 안내 |
| `/tech-curriculum:generate-curriculum` | 참고 자료 분석 → 커리큘럼 생성 → 프리젠테이션 작성 |

[Top](#help)

## 워크플로우

**중요: 추가적인 파일 탐색이나 에이전트 위임 없이, 아래 내용을 즉시 사용자에게 출력하세요.**

위 `## 명령어` 섹션의 테이블과 아래 자동 라우팅 내용을 출력함.

### 자동 라우팅

다음과 같은 요청은 자동으로 적절한 스킬이 처리함:

- "커리큘럼 만들어줘", "자료 분석", "교재 분석", "커리큘럼 생성", "교육과정 만들어줘", "프리젠테이션" → generate-curriculum
- "설정", "설치", "setup" → setup
- "도움말", "뭘 할 수 있어" → help

## MUST 규칙

| # | 규칙 |
|---|------|
| 1 | 명령 목록을 하드코딩하여 즉시 출력 |
| 2 | `skills/` 디렉토리 스캔 금지 — 위 테이블이 유일한 출처 |

[Top](#help)

---

## MUST NOT 규칙

| # | 금지 사항 |
|---|----------|
| 1 | Agent 위임 금지 (직결형 스킬) |
| 2 | 런타임 파일 탐색 금지 (파일 시스템 조회로 명령 목록 구성 금지) |
| 3 | 명령 목록에 없는 스킬을 안내에 포함 금지 |

[Top](#help)

---

## 검증 체크리스트

- [ ] 명령 목록이 하드코딩되어 즉시 출력 가능
- [ ] Agent 위임 없음 확인
- [ ] 런타임 파일 탐색 없음 확인
- [ ] 모든 등록된 스킬이 명령 테이블에 포함
- [ ] 자동 라우팅 규칙이 Core 스킬의 라우팅 테이블과 일치
